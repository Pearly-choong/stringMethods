substr()method

  • Similar to slice()
  • The difference is that the second parameter specifies the length of the extracted part.
  • Returns the specified number of characters from the specified index from the given string.

Syntax:

str.substr(start, length)

Arguments:

1st argument : start - defines the starting index from where the substring is to be extracted from the base string.
2nd argument : length - defines the number of characters to be extracted starting from the start in the given string.
If the second argument to the function is undefined, then all the characters from the start till the end of the length is extracted.

Return value:

This function returns a string that is the part of the given string.
If the length is 0 or negative value then it returns on empty string.

replace()method

  • Replace a specified value with another value in a string
  • Does not change the string it is called on. It returns a new string.
  • By default, it replace only the first match.
  • This function is case sensitive.

Syntax:

str.replace(substr | regexp , newSubstr)

Arguments:

1st argument : substr | regexp - can be a string or the regular expression ( patterns used to match character combinations in strings ).
2nd argument: newSubstr - is a string which will replace the content of the given string. The replacement also can be a function to be called for each match.

Return value:

It returns a new string with replaced items.
